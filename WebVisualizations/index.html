    • Summary: Latitude vs. X

    The Latitude vs X project aims to answer the question "What is the weather like as we approach the equator?" Through this challenge, max temperature, humidity, cloudiness, and wind speed were analyzed at various latitudes across the globe.

    To accomplish this analysis, data was pulled from the CityPy library to assemble a random set of over 500 cities. Then, the OpenWeatherMap API was used to identify weather data for the cities dataset. After generating the dataset, Matplotlib was used to plot the weather data against latitude. This site provides the source data and visualizations created as part of this analysis, as well as explanations and descriptions of any trends and correlations observed.
